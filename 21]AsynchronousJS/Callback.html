Callbacks

Callback matlab ek aisa function jo doosre function ke andar argument ki tarah pass kiya jata hai aur baad mein chalaya jata hai jab koi kaam khatam ho jata hai. Jaise ki server se data laane ke baad ya file read karne ke baad.

Callbacks are functions that are passed as arguments to other functions and are executed after the completion of a certain operation. They are commonly used in asynchronous programming to handle the results of operations that take time to complete, such as fetching data from a server or reading a file.


        // Basic Callback syntax
        function abcd (a,b){
            b();
        }
        abcd(1,function(){alert("Callback Chal Gaya")})
