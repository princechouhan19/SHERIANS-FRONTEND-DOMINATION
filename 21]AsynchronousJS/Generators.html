Generators

Generators aise special functions hain jinhe pause aur resume kiya jaa sakta hai. Inhe define karne ke liye function* syntax use hota hai.

Generators are special functions that can be paused and resumed. They are defined using the function* syntax. 


        function* generator() {
            console.log('printing 1');
            yield 1; // pauses the function and returns 1
            console.log('printing 2');
            yield 2;
            console.log('printing 3');
            yield 3;
        }
        const ans = generator(); // ans is a generator object
        console.log(ans.next()); // {value: 1, done: false}
        console.log(ans.next());
        console.log(ans.next());
        console.log(ans.next()); // {value: undefined, done: true}
